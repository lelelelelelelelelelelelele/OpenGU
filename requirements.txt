# OpenGU / GULib runtime dependencies

# torch is NOT pinned here because it must be installed against a specific CUDA
# version via the PyTorch index URL. See `GULib-master/SERVER_RUNBOOK.md` §A

# torch_scatter / torch_sparse must match torch+CUDA version (use the wheel
# index https://data.pyg.org/whl/torch-<ver>+<cu>.html).

# requirements (cupy / pymetis / memory-profiler / numba / gensim / lightgbm /
# Cython). All confirmed real runtime deps via codebase grep + local env scan.
