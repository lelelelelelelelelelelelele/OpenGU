# 附录：方法分类与机制说明

## 方法分类表

| 类型 | 方法 | 机制描述 | 攻击敏感性 |
|------|------|----------|------------|
| IF-based | GIF | 基于梯度的影响函数 | 中等（IF/IM 有效） |
| Learning-based | GNNDelete | 基于节点嵌入的影响函数 | **高**（IM 最有效） |
| Shard-based | GraphEraser | 图分区 + 子模型聚合 | **免疫**（反效应） |
| Shard-based | GUIDE | 图分区 + MixUp 修复 | **免疫**（反效应） |

## 攻击策略说明

| 策略 | 原理 | 适用场景 |
|------|------|----------|
| Random | 随机选择节点 | 基线 |
| Degree | 选择高度数节点 | 图结构重要性 |
| PageRank | 选择高 PageRank 节点 | 传播影响力 |
| TracIn | 梯度方向影响函数 | 参数层面重要性 |
| IM | 影响力最大化 (CELF) | 传播影响力 |
| Hybrid | TracIn + IM 融合 | 双重重要性 |

## 评估指标说明

| 指标 | 说明 | 理想值 |
|------|------|--------|
| F1 Drop | 遗忘后 F1 变化 | > 0 (攻击成功) |
| Retrain Gap | 近似 vs 精确重训差距 | 接近 0 |
| MIA AUC | 成员推断攻击效果 | < 0.5 (安全) |
| Collateral | 附带损伤程度 | 接近 0 |
