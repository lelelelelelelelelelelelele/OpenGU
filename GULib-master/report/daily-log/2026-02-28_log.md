## 🔍 关键发现

### 1. GNNDelete 脆弱性进一步确认（多 seed 验证）
- seed=2024 下 im_v4 F1 Drop = 0.2712（27.12%），是 random 的 3.87x
- seed=1337 下 im_v4 F1 Drop = 0.1974（19.74%）
- 跨 seed 一致性强，结论稳健

### 2. 方法族差异模式稳定
- **Learning-based (GNNDelete)**: 高脆弱（F1 Drop 7-27%），策略选择影响显著
- **IF-based (GIF)**: 中等脆弱（F1 Drop 0.4-4.2%），hybrid_v4 和 im_v4 最有效
- **Shard-based (GraphEraser)**: 负 F1 Drop（-0.9% ~ -10.3%），Shard 保护效应持续
- **GUIDE**: F1 Drop = 0（源库 Bug，已标记为无效数据）
